# 🚀 大拇哥积分商城 - 快速启动

## 一、环境准备

确保已安装：
- Python 3.8+
- Node.js 16+
- MySQL 8.0+

**检查环境**：双击运行 `检查环境.bat`

---

## 二、数据库初始化

### 方法1：手动执行（推荐）

```bash
mysql -u root -p < database\init.sql
```

### 方法2：使用管理工具

双击运行 `管理工具.bat`，在"数据库管理"选项卡操作。

---

## 三、配置数据库密码

### 方法1：使用管理工具（推荐）

1. 双击运行 `管理工具.bat`
2. 打开"配置管理"选项卡
3. 输入 MySQL 密码
4. 点击"保存配置"

### 方法2：手动编辑

编辑 `backend/config.py` 第 14 行，或创建 `backend/.env` 文件。

---

## 四、启动服务

### 方法1：使用管理工具（推荐）

双击运行 `管理工具.bat`，在"服务管理"选项卡：
- 点击"启动后端服务"
- 点击"启动前端服务"

### 方法2：手动启动

```bash
# 启动后端
双击 backend\run.bat

# 启动前端
双击 frontend\run.bat
```

## 常见问题

### 登录失败？
运行 `管理工具.bat` → "数据库管理" → "重置所有测试用户密码"

### 数据库连接失败？
检查 `backend/.env` 中的密码是否正确
